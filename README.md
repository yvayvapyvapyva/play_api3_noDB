# 🧭 NAVIGATOR — VK Mini Apps

Веб-приложение для автоматического ведения по учебным автомобильным маршрутам с голосовыми подсказками.

## 🚀 Быстрый старт

### Локальное тестирование

```bash
# Используем любой локальный сервер
npx http-server -p 5501
# или
python3 -m http.server 5501
```

### Размещение в VK

1. Загрузите файлы на HTTPS-хостинг (GitHub Pages, Vercel, Netlify)
2. В [VK Developers](https://dev.vk.com/) создайте мини-приложение
3. Укажите URL вашего приложения

## 🎯 Как использовать

### Запуск приложения

1. Откройте `index.html` в браузере или VK Mini Apps
2. Нажмите на кнопку **Меню** (слева вверху)
3. Выберите маршрут из списка или введите `ID-название`
4. После загрузки маршрута автоматически начнётся авто-ведение
5. Кнопка **Play** (слева вверху) запускает ведение повторно

### Функционал

- **Меню** — выбор и загрузка маршрута
- **Play** — запуск авто-ведения по маршруту
- **HUD** (внизу) — текущая команда и кнопка воспроизведения аудио
- **Wake Lock** — экран не гаснет во время ведения

## 📝 Примечания

- Геолокация требует HTTPS или localhost
- Wake Lock API может не работать в некоторых вебвью
- Для работы необходим Yandex Maps API Key (уже включён)
